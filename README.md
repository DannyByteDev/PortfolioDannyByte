# DannyByte Portfolio

Portfolio personal de DannyByte - Full Stack Developer, construido con Next.js, TypeScript, Tailwind CSS y Framer Motion.

## Tech Stack

- **Framework**: Next.js 16 (Static Export)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4 + Custom CSS
- **Animations**: Framer Motion + tsparticles
- **Deployment**: GitHub Pages

## Features

- Animaciones avanzadas (glitch text, typewriter, 3D cards, parallax)
- Terminal de código animada en el hero
- Toggle de tema claro/oscuro
- Formulario de contacto con FormSubmit
- Sección de timeline de experiencia
- Testimonials carousel
- Modal de detalle de proyectos
- Cursor personalizado con física de springs
- Fondo de partículas interactivas
- Responsive design (mobile-first)

## Development

```bash
# Install dependencies
bun install

# Run development server
bun run dev

# Build for production (static export)
bun run build
```

The static output will be in the `out/` directory.

## Deploy

El proyecto se despliega automáticamente en GitHub Pages via GitHub Actions cuando haces push a la rama `main`.
